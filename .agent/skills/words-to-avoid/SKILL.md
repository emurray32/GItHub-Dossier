# Words to Avoid — Banned BDR Vocabulary

These words and phrases kill deals. They signal "salesperson" instead of "peer." They trigger spam filters. They make the reader's eyes glaze over. None of these should appear in any email, ever.

## Category 1: Salesy Greetings (BANNED)

| Banned Phrase | Why It's Banned |
|---|---|
| "I hope this email finds you well" | #1 signal this is a cold email |
| "Hope you're doing well" | Same — instant delete |
| "I hope you are having a great week" | Nobody writes this to colleagues |
| "Hi there" | Generic, impersonal |
| "Hello" | Too formal for peer-to-peer |
| "Dear {{first_name}}" | 2005 called |
| "Good morning/afternoon" | Timezone-ignorant, formal |
| "Greetings" | Alien landing energy |

**Instead:** Always `Hey {{first_name}},` — nothing else.

## Category 2: Empty Openers (BANNED)

| Banned Phrase | Why It's Banned |
|---|---|
| "I came across your company..." | Generic, says nothing specific |
| "I've been following your work..." | Creepy if not specific |
| "I was impressed by..." | Flattery without substance |
| "I'm reaching out because..." | Announces the cold email |
| "I wanted to introduce myself..." | Nobody cares who you are yet |
| "My name is X and I work at..." | Company intro = instant skip |
| "We're a leading provider of..." | Self-praise, zero value |
| "I noticed your company is growing..." | Vague, says nothing |
| "I see you're working on exciting things" | Meaningless |

**Instead:** Open with the specific signal: "Noticed you added `react-i18next` to `main-app`."

## Category 3: Marketing Speak (BANNED)

| Banned Word/Phrase | Why It's Banned |
|---|---|
| "comprehensive" | Nobody wants comprehensive. They want simple. |
| "robust" | Meaningless adjective |
| "cutting-edge" | Every company says this |
| "state-of-the-art" | See above |
| "best-in-class" | Self-awarded title |
| "industry-leading" | Same |
| "world-class" | Same |
| "innovative" | So overused it means nothing |
| "revolutionary" | Nothing is revolutionary |
| "game-changing" | Nothing is game-changing |
| "next-generation" | What generation? |
| "seamless" | No integration is seamless |
| "end-to-end" | Vague scope claim |
| "holistic" | MBA word, not dev word |
| "synergy" | Just... no |
| "leverage" (as a verb) | Use "use" instead |
| "utilize" | Use "use" instead |
| "optimize" | Overpromise word |
| "empower" | Corporate fluff |
| "enable" | "lets you" is better |
| "transform" | Promise you can't make in Email 1 |
| "unlock" | Marketing, not engineering |
| "streamline" | Weak verb, vague claim |
| "elevate" | Meaningless |
| "accelerate" | Only if quantified |
| "supercharge" | Never, under any circumstances |

**Instead:** Use plain, specific verbs: "automates," "removes," "handles," "plugs into."

## Category 4: Pressure & Urgency (BANNED)

| Banned Phrase | Why It's Banned |
|---|---|
| "Limited time" | Fake scarcity |
| "Act now" | Infomercial energy |
| "Don't miss out" | FOMO manipulation |
| "Before it's too late" | Manufactured urgency |
| "Exclusive offer" | There's no offer |
| "Special pricing" | Don't discuss pricing in cold email |
| "I only have a few slots" | Lie, and they know it |
| "This week only" | Fake deadline |

**Instead:** Let the signal create natural urgency: "Teams at this stage usually..." implies timing without pressure.

## Category 5: Needy Follow-Up Language (BANNED)

| Banned Phrase | Why It's Banned |
|---|---|
| "Just following up" | #1 most deleted phrase in B2B email |
| "Bumping this" | Aggressive, entitled |
| "Circling back" | Adds no value |
| "Checking in" | Empty, annoying |
| "Did you get my last email?" | Needy, desperate |
| "Haven't heard back" | Guilt trip |
| "I know you're busy, but..." | Patronizing |
| "Sorry to bother you" | Undermines your value |
| "Just wanted to touch base" | Zero-content phrase |
| "Putting this back on your radar" | You're not their boss |
| "As I mentioned previously" | Lazy — bring something new |
| "Per my last email" | Passive-aggressive |

**Instead:** Every follow-up opens with a NEW insight or angle. See `follow-up-emails` skill.

## Category 6: Phrase-Specific Banned Topics

| Banned Topic | Why |
|---|---|
| "high quality translations" | Devs care about process, not linguist quality |
| "professional linguists" | Irrelevant to the dev buyer |
| "translation quality" | Not a first-touch topic |
| Competitor names (Lokalise, Crowdin, Transifex) | Never badmouth; never compare |
| Pricing | Never in cold outreach |
| Case studies by name | Don't name-drop customers without permission |
| ROI statistics you can't source | Don't fabricate numbers |
| "We work with companies like X" | Name-dropping without permission |

**Instead:** Talk about what Phrase DOES (automation, GitHub Sync, CI/CD), not what it IS (a TMS, a platform).

## Category 7: Formatting Violations (BANNED)

| Banned Element | Why |
|---|---|
| Exclamation marks (!) | Salesy, over-eager. Zero allowed. |
| Emoji | Unprofessional in cold B2B outreach |
| Bold text | Looks like marketing email |
| Italic text | Same |
| Bullet points in Email 1 | Too structured for a peer email |
| Images or logos | Spam filter trigger |
| HTML formatting | Plain text only |
| Multiple links | One link max, and only in Email 2+ |
| Long paragraphs (3+ sentences) | Wall of text = not read |
| ALL CAPS words | Shouting |

## Category 8: Weak Closings (BANNED)

| Banned Phrase | Why | Instead |
|---|---|---|
| "Let me know" | Passive, no specific ask | "Worth a look?" |
| "Feel free to reach out" | Puts all burden on them | "Open to a quick look?" |
| "I'd love to chat" | Too eager, about YOUR wants | Ask about THEIR interest |
| "Would you be open to a call?" | Too much commitment | "Worth exploring?" |
| "Can we schedule a meeting?" | Way too much for Email 1 | Ask for interest, not time |
| "Looking forward to hearing from you" | Assumptive, corporate | Just sign with name |
| "Best regards" | Too formal | `{{sender_first_name}}` only |
| "Cheers" | Tries too hard to be casual | `{{sender_first_name}}` only |
| "Thanks" | Thanks for what? | `{{sender_first_name}}` only |
| "Best" | Corporate sign-off | `{{sender_first_name}}` only |

## The Quick Test
Before sending any email, scan for EVERY word in this file. If any appear, rewrite that sentence. No exceptions.
